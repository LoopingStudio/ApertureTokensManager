### Nouveautés de la version 1.0.6

#### Graphe de dépendances

Nouvel onglet **Graphe** pour visualiser les chaînes de dépendance entre vos tokens.

- **Vue en colonnes** : Primitives, Niveau 1 et Niveau 2 organisés de gauche à droite
- **Connexions bézier** : Les liens entre tokens sont tracés en courbes attachées aux bords des cartes
- **Survol interactif** : Survolez un token pour mettre en surbrillance toute sa chaîne de dépendance
- **Sélection** : Cliquez sur un token pour afficher son chemin complet (ancêtres → descendants)
- **Isolation** : Isolez les connexions d'un token pour masquer le reste du graphe, avec animation
- **Recherche** : Filtrez les tokens par nom ou chemin directement dans le graphe
- **Zoom** : Zoomez avec le trackpad pour naviguer dans les grands graphes
- **Filtre Utility** : Masquez les tokens du groupe Utility pour réduire le bruit
- **Détail couleur** : Cliquez sur un aperçu couleur pour voir le code hexadécimal et la primitive associée
- **Sélecteur brand/thème** : Basculez entre Legacy/New Brand et Light/Dark pour voir les résolutions
